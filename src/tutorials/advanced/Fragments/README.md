# Fragments

- group the set of DOM elements without extra node

## usage

```jsx
// default
<React.Fragment>
    // children component/dom goes here
</React.Fragment>

// shorthand
<></>
```

## example

```jsx
<React.Fragment>
  <h1>Header</h1>
  <h2>Subtitle</h2>
  <p>Fragment used to group the set of children without extra node</p>
</React.Fragment>
```

```jsx
<>
  <h1>Header</h1>
  <h2>Subtitle</h2>
  <p>Fragment used to group the set of children without extra node</p>
</>
```
