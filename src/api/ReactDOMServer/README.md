# ReactDOM Server

- render components to static html markup
- mainly used in the `node server`

## Usage

- SSR support
- SEO
- Fast page loading

## API's

- for `server` and `browser` rendering

  - **renderToString()**
  - **renderToStaticMarkup()**

- for only `server` end

  - **renderToNodeStream()**
  - **renderToStaticNodeStream()**
